Data Protection

All the inventory data you add will be saved in your browser temporarily. None of your data will be sent to other websites or servers.

User Privacy

  • We don't collect any of your personal information
  • All data stays only on your own device
  • Data will be removed when you close the browser

Security Measures

  • We check user input to make sure data is valid
  • Each item uses a unique ID to avoid conflict
  • The page layout works on phones and computers
